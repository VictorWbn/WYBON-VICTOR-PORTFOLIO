WINTER 1944

Technologies : Java

Durant le premier semestre du BUT Informatique, j'ai développé le célèbre jeu du "Chameau" (GitHub Chameau). Pour me familiariser davantage avec le langage Java, j’ai créé une version alternative du jeu, centrée sur la Seconde Guerre mondiale. Cette version inclut plusieurs améliorations, notamment un système de combat et un inventaire plus complet, offrant une expérience enrichie par rapport au jeu d’origine.

Le code actuel comporte encore des bugs, car il s'agit d'une première version basée sur un script. Je travaille actuellement sur une mise à jour orientée objet pour rendre le code plus robuste et évolutif.
